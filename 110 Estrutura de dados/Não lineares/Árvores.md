# Árvores

Árvores é uma estrutura de dados usada para organizar e representar dados, de forma não linear, em uma estrutura hierárquica.

É provavelmente das estruturas de dados mais complexa a mais importante, já que elas estão em todo lugar, por exemplo o sistema de arquivos em computadores é inteiro estruturado como árvore, a DOM de uma página HTML é uma árvore e etc, tudo que representa uma forma hierárquica é uma árvore.

Uma árvore possui *nodes*, onde o *node* mais ao topo se chama *root*, e cada outro *node* pode ter um ou mais *child nodes*, consequentemente para esses *child nodes*, o *node* anterior é chamado de *parent node*.

As árvores podem ser classificadas com base no número de *childs nodes* que cada *node* tem, mas geralmente elas são separadas entre dois tipos principais:

- **Árvore binária:** Cada *node* tem no máximo 2 *child nodes*
- **Árvore genérica:** Cada *node* pode ter *n child nodes*.
