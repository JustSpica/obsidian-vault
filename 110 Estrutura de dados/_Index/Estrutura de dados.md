São formas de organizar e armazenar informações na memória do computador para que possam ser usadas de maneira eficiente e prática para a compreensão humana. Elas definem a maneira como os dados são acessados, manipulados e armazenados. 

**NÃO SÃO TIPOS DE DADOS, SÃO CONCEITOS.**

Existem dois tipos de estrutura de dados:

- **Linear:** São organizados em sequência na memória, formando uma cadeia de dados sequencial onde o acesso e a manipulação dos dados geralmente seguem essa ordem.
- **Não linear:** Não são organizados em sequência na memória. Elas utilizam a memória de forma mais eficiente e os dados ficam dispersos.
